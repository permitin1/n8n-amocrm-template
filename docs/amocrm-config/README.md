# Конфиг amoCRM

Файлы в этой папке описывают конфигурацию конкретного аккаунта amoCRM.
Используются Claude Code при создании и обновлении воркфлоу.

## Когда обновлять

- Добавили новое поле в сделки/контакты/компании
- Добавили/переименовали этап воронки
- Добавили нового сотрудника
- Настроили новый вебхук
- Добавили/удалили теги

## Как обновить

```bash
node scripts/init-amocrm-config.js
```

Скрипт перезапишет все файлы кроме salesbots.md.

## salesbots.md

Заполняется вручную — у amoCRM нет публичного API для сейлзботов.
Как найти ID бота: Настройки → Salesbot → открыть бота → ID в URL браузера.
