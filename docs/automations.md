# Автоматизации

Актуальное описание всех работающих автоматизаций проекта.
Обновляется автоматически при создании/изменении BT-задач.

---

## N8N воркфлоу

### BT-001 Выбор мессенджера
- **Тип:** both (n8n + service)
- **Триггер:** amoCRM webhook → message events
- **Что делает:** Парсит deeplink из сообщения бота (Telegram/MAX), находит или создаёт контакт/сделку, привязывает неразобранное, ставит чекбокс мессенджера на контакте, создаёт задачу менеджеру
- **Entry-роутер:** webhook-message.json

## Веб-сервис

### BT-001 Выбор мессенджера
- **Роуты:** GET `/messenger`, GET `/landing`
- **Что делает:** Страница выбора мессенджера (TG/MAX) с deeplink-ами. Три сценария: по contact_id, по телефону+UTM, мини-лендинг с полем ввода. QR-коды на десктопе.
